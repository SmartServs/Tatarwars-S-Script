## 📌 وصف المشروع

إصدار جديد من سكربت حرب التتار الكلاسيكي **(S6 LTS)**، وهو النسخة النهائية من سلسلة الإصدارات الكلاسيك.

يقدم السكربت تجربة لعب كلاسيكية سريعة وخفيفة مع تحسينات قوية في الأداء، الحماية، وتنظيم الكود، مع الحفاظ على روح اللعبة الأصلية.
